- Vim-like operations
- Modes:
  - bottom-line
  - normal
  - insert
- Windows
```
________________________
|       |               |
|       |               |
|       |               |
|       |               |
|_______|_______________|
```

- Operation
  - `CTRL + W` shift between subwin
  - `i` enter insert mode
  - `ESC` enter normal mode
  - `:` enter bottom line mode

  - In bl mode:
    - `signin`
    - `signout`
    - `signup`
    - `addfriend`
     - `delfriend`
